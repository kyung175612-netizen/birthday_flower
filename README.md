* 작업한 순서: HTML 구조 설계 -> CSS -> JS 구현
* 발생한 문제 상황 & 해결 방법:
    * 팝업창 띄울 때 배경이 전체에 적용되어 앱에 맞게 조절
    * 팝업창 위치가 맞지 않아 여러 번 시도하며 설정
    * z-index:(가로x축,세로y축) z축은 깊이를 말함(팝업창을 팝업배경보다 앞으로 나오도록 함)
    *앱 배경와 팝업창 배경이 서로 위치가 맞지 않아 팝업창 배경에 margin:0 auto;를 주었으나 바뀌지 않음. 그래서 main에 position: relative; 입력하여 조정
    * 배경을 조정하였더니 팝업창이 옮겨져 다시 팝업창 위치 설정.
    * 팝업창의 배경에 투명도를 주었는데 너무 어둡거나 밝아 조정.
    * js에서 팝업창 숨기기를 한 후에, '나의 꽃과 꽃말 만나기' 버튼 클릭해보았더니 팝업창이 나오지 않음. -> 버튼에 onclick, 스크립트에 defer를 입력하지 않아서 팝업창이 나오지 않음. 입력 후에는 실행됨.
    * id는 #으로 입력해야 하는데 .으로 입력해서 '12'가 보이지 않음.->#으로 변경함.
